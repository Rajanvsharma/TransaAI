# Transa TMS AI

Complete deployable Next.js landing website for Transa TMS AI.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build for production

```bash
npm run build
npm start
```

## Deploy

Upload this folder to GitHub and import it into Vercel, or drag the project into Vercel.

## Working links

All nav links scroll to real sections. Demo/contact buttons open the demo section, email demo request, or phone sales link.
